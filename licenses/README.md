This directory contains third-party license texts generated from the Gradle OSS Licenses plugin on 2026-01-09.
Files:
- third_party_licenses.txt : concatenated license text for all dependencies (copied from app/src/main/res/raw)
- third_party_license_metadata.json : metadata for licenses
- components/ : individual component license texts
- fonts/ : font-specific license files extracted from third_party_licenses
